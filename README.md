# ChatGPT Review Analysis

## Project Objective
The objective of this project is to analyze customer reviews of ChatGPT to understand overall user sentiment, subjectivity, and commonly discussed features. The analysis helps identify positive feedback patterns as well as areas of user concern.

---

## Problem Statement
How do users perceive ChatGPT based on their reviews, and what sentiment trends and key aspects emerge from user feedback?

---

## Dataset Overview
The dataset contains textual customer reviews with relevant metadata.  
Key fields include:
- Review Text
- Rating (if available)
- Review Date

---

## Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- TextBlob (Sentiment Analysis)
- Matplotlib & Seaborn (Data Visualization)

---

## Data Preparation
- Loaded and inspected the dataset
- Standardized column names
- Handled missing review text values
- Converted numeric fields to appropriate data types
- Ensured the dataset was clean and ready for analysis

---

## Sentiment Analysis
- Calculated sentiment **polarity** to classify reviews as positive, neutral, or negative
- Calculated **subjectivity** to measure opinion strength
- Created sentiment categories based on polarity scores
- Compared sentiment trends across the dataset

---

## Text Analysis
- Analyzed frequently occurring words in reviews
- Identified commonly discussed positive features
- Highlighted recurring themes in user feedback

---

## Exploratory Data Analysis (EDA)
- Distribution of sentiment polarity
- Distribution of subjectivity scores
- Sentiment category comparison using bar charts
- Keyword frequency analysis

---

## Visualizations
- Bar chart showing sentiment distribution
- Histogram of subjectivity scores
- Frequency plots for commonly used keywords

---

## Key Insights
- Most user reviews express positive sentiment
- Subjectivity scores indicate that reviews are largely opinion-based
- Users frequently mention usability and response quality
- Negative sentiment is limited but highlights areas for improvement

---

## Conclusion
This project provides a clear overview of how users perceive ChatGPT based on review data. The insights can help improve user experience and guide future enhancements.
